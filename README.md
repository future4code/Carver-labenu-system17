## LabenuSystem:

As funcionalidades básicas são:

→ Criar turma;
→ Busar turmas ativas;
→ Mudar turma de módulo ;

→ Criar estudante;
→ Buscar estudantes através do nome;
→ Mudar estudante de turma;

→ Criar docente;
→ Buscar todas as pessaos docentes;
→ Mudar docente de turma;
